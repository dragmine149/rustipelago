# Rustipelago

A full fletched application meant to be a drop-in replacement for the original application provided by [https://archipelago.gg/]

## Why?
Because i've had issues with the official one, namily
- slow startup
- No copy/paste
- Having to remember the slot names
- Probably some other minor annoyances...

hence i just decided to make my own.

### Why rust?
Because rust is my primary language, thats all. Right tool for the job? idk. If you did it right, i'm guessing you could even get the python version to preform better.

## Contributions
Comtributions are welcome, however most of the effort will in-fact lie on other sources. We just provide the interface and let everyone else actually do the smart part of the worlds.
